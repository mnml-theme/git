[mnml] [`git`]
==============
A **m**i**n**i**m**a**l** [theme] for [`git`].

Install
-------
with _[Home`brew`]_:
~~~ sh
brew install mnml-theme/install/mnml-git
~~~
or `brew bundle install` from a _[Brewfile]_:
~~~ rb
tap "mnml-theme/install"
brew "mnml-git"
~~~
